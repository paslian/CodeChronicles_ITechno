Apa itu HTML?

Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web dan aplikasi web.

Definisi HTML

HTML adalah bahasa markup yang digunakan untuk menyusun struktur konten di web.

Struktur Dasar HTML

<!DOCTYPE html>
<html>
<head>
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Ini adalah heading</h1>
  <p>Ini adalah paragraf</p>
</body>
</html>

Penggunaan HTML dalam Pengembangan Web

HTML digunakan bersama CSS dan JavaScript untuk membuat halaman web yang interaktif dan menarik.
